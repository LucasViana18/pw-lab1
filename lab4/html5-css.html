CSS

CSS (Cascade Style Sheet) permite controlar a aparência de elementos HTML.

Seletores de CSS
Seletor Foi usado? Se sim, onde e como funciona?
#id
 Foi usado nesta página atual. Este seletor seleciona o elemento com o respetivo id. Exemplo: #firstname
.class
 -. Este seletor seleciona todos os elementos da respetiva classe. Exemplo: intro
element.class
 -. Este seletor seleciona um específico elemento da respetiva classe. Exemplo: p.intro
*
 -. Este seletor seleciona todos os elementos. Exemplo: *
element
 -. Este seletor seleciona um elemento específico. Exemplo: p
Combinadores de seletores de CSS
Combinador de seletor Foi usado? Se sim, onde e como funciona?
(espaço)
 -. Todos os elementos que estão dentro do primeiro elemento escrito são selecionados. Exemplo: div p
>
 -. Todos os elementos que estão dentro do primeiro elemento escrito e sejam filhos e não descendentes posteriores (netos ou outros descendentes) são selecionados. Exemplo: div > p
~
 -. Todos os elementos que estão colocados depois do primeiro elemento escrito são selecionados. Exemplo: div ~ p
+
 -. Todos os elementos que estão imediatamente colocados depois do primeiro elemento escrito são selecionados. Exemplo: div + p
,
 Foi usado nesta página atual. Todos os elementos escritos são selecionados. Exemplo: div, p
